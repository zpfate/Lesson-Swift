## [5. 最长回文子串](https://leetcode.cn/problems/longest-palindromic-substring/)



给你一个字符串 `s`，找到 `s` 中最长的 回文 子串。

* 如果字符串向前和向后读都相同，则它满足 **回文性**。

* 子字符串** 是字符串中连续的 **非空** 字符序列。

 

**示例 1：**

```
输入：s = "babad"
输出："bab"
解释："aba" 同样是符合题意的答案。
```

**示例 2：**

```
输入：s = "cbbd"
输出："bb"
```

 

**提示：**

- `1 <= s.length <= 1000`
- `s` 仅由数字和英文字母组成
